# Embed an Image

Store images in the assets folder and then embed them using normal markdown syntax.

```markdown
![](../../assets/signet-architecture.png)
```

Renders as:

![](../../assets/signet-architecture.png)
![](../../assets/robots.svg)
![](../../assets/testing_pyramid.svg)
![](../../assets/trains_1.svg)
![](../../assets/trains_2.svg)
